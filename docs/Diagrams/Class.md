# Глоссарий

1. **Client** – класс, представляющий клиента CRM-системы.
2. **Order** – класс, представляющий заказ.
3. **User** – класс, отвечающий за учетные записи и вход в систему.
4. **Activity** – класс, описывающий действия пользователя в системе.
